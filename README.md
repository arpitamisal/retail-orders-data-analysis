# Retail Orders Data Analysis (Python)

This project analyzes retail order data to identify regional sales performance, top-selling products, and monthly sales trends using a data processing pipeline and analytical queries.

## Tech Stack
- Python  
- Pandas & NumPy  
- SQL Server  
- Matplotlib & Seaborn  
- Jupyter Notebook  
- Kaggle API  

## Architecture
- Dataset downloaded from Kaggle using the Kaggle API  
- Data cleaning and preprocessing performed in Python using Pandas  
- Processed data loaded into SQL Server  
- SQL queries used for sales analysis and aggregations  

## Key Work
- Data cleaning and preprocessing of retail order dataset  
- Aggregated total sales by region to compare regional performance  
- Identified top-performing products within each region using ranking techniques  
- Analyzed monthly sales trends across multiple years  
- Built visualizations to highlight sales distribution and performance insights  

The project demonstrates an end-to-end data workflow from dataset acquisition and preprocessing to SQL-based analytics and visualization.
